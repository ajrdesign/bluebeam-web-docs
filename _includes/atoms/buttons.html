Buttons

Buttons are classes are compounding so basically all buttons are exactly the same and you can swap out colors and sizes by adding or subtracting classes. This means there can be a large variety of buttons without a huge amount of repeated code. They are also 100% CSS so there is no requests for images needed.

These classes can be applied to any element to act as a "button" for example:
