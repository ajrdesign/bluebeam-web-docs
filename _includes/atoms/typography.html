Typography

Below outlines the typefaces used for Bluebeam and how they can be manipulated with CSS.

Typefaces

Avenir

Our primary typeface. This is typically used in Black weight for headlines and a light weight for body copy and subheadings.

Proxima Nova

This is a secondary typeface that can be used as an accent. It's generally used as a subheading in a different color, often in all caps. You can access this typeface by adding .subheading to a heading tag.

Headings

h1-6 are the most commonly used but we have larger sizes as it's occaionally useful to have something larger for hero images. You can also add .space-bottom to add a little bit more margin to the bottom of headings. Additionally adding .flush-margin-b will remove additional padding below any heading. See the helpers section for more modifying classes like these.

Body Copy

Extended body copy should not be set wider than 600px. This keeps the character count in each line less than 90 which is close to ideal for good readibility.

Lead Copy

This is to denote a introduction paragraph or call out a section. This should always be used to introduce any large body copy. The technique helps ease people into longer copy by making the first paragraph easily scanable while the rest of the copy is at an appropriate line length for long reading.
